## 1. Executive Summary

Whites Printing Services LLC is a full-service commercial print shop located near National Paints Metro Station, Deira, Dubai. Operating for 15+ years with a 7-person specialist team and flagship Canon imagePRESS C10000VP technology, Whites serves a diverse client base across 16 industry verticals — from hotels and hospitals to real estate firms and government entities.

The whites.me website is being built to transform Whites' digital presence from a basic online brochure into a lead-generation engine. The core business objectives are:

| Objective | Target Metric |
|-----------|--------------|
| Generate qualified inbound leads | ≥50 quote form submissions per month (year 1) |
| Rank locally for printing-related searches | Top 3 on Google for 10+ target keywords (12 months) |
| Convert visitors into quote requests via WhatsApp | ≥30% WhatsApp CTR from site visitors |
| Establish credibility with UAE corporate/SME clients | ≤35% bounce rate; ≥2 min avg. session duration |
| Reduce friction in the enquiry process | ≥15% form-to-lead conversion rate |

**Success Metrics (KPIs):**
- [ ] Lead form submissions (monthly count + conversion rate)
- [ ] WhatsApp click-through rate (clicks ÷ sessions)
- [ ] Organic traffic growth (month-over-month, GA4)
- [ ] Bounce rate (target ≤35%)
- [ ] Average session duration (target ≥2 min)
- [ ] Google Business Profile insights (direction requests, calls, website clicks)
- [ ] Pages indexed in Google Search Console
- [ ] Core Web Vitals pass rate (LCP <2.5s, FID <100ms, CLS <0.1)

## 3. Feature Specifications

### F1: Request a Quote Form

| Field | Detail |
|-------|--------|
| **Feature ID** | F1 |
| **Feature Name** | Request a Quote Form |
| **Description** | Multi-field web form allowing visitors to request a printing quote. Supports file upload (artwork/print files), service selection, quantity, and delivery requirements. Submission triggers email notification to Whites team + WhatsApp follow-up. |
| **Priority** | **High** |
| **Primary Personas** | Sarah (corporate), Khalid (SME), Maya (agency) |

**User Story:**
> As a visitor, I want to upload my artwork and describe my print requirements so that I can receive an accurate quote without needing to call.

**Acceptance Criteria:**

- [ ] Form includes fields: Full Name, Phone Number (required), Email, Company Name, Service Selection (dropdown of 15 services), Quantity, File Upload (max 25MB, accepted formats: PDF, AI, EPS, CDR, PSD, JPG, PNG, TIFF), Description/Comments, Delivery Required (checkbox), Installation Required (checkbox)
- [ ] File upload supports drag-and-drop and click-to-browse; shows upload progress
- [ ] Form submission triggers:
  - Email notification to Whites sales team with all form data + file attachment links
  - WhatsApp auto-reply to submitter with quote acknowledgement and expected response time
- [ ] CAPTCHA or honeypot spam protection implemented
- [ ] Form data stored in CMS/backend for lead tracking
- [ ] Mobile: input font size ≥16px (prevents iOS zoom); vertically stacked layout
- [ ] Success message displayed on submission: "Thanks! We'll respond via WhatsApp within 2 hours"
- [ ] Form appears on: Dedicated "Get a Quote" page, each service sub-page (CTA section), Contact page
- [ ] File upload uses secure, scalable storage (not server-local; use cloud storage or CDN)

## 6. Out of Scope (Phase 1)

The following features are explicitly excluded from Phase 1 development:

| # | Feature | Rationale |
|---|---------|----------|
| 1 | **E-commerce / Cart / Payment Gateway** | Quote-based model is the approved workflow. No cart, checkout, or payment processing at this stage. |
| 2 | **User Authentication / Accounts** | No login, no user dashboard, no order history portal. All communication via WhatsApp/phone. |
| 3 | **Multi-Language Support** | Arabic version deferred. English-only for Phase 1. Arabic may be added in Phase 2 if analytics justify investment. |
| 4 | **Corporate Gifts Full Order Flow** | Soft enquiry page only (contact us to discuss). Full product catalogue, pricing, and ordering deferred until trade supplier relationships are confirmed. |
| 5 | **Blog / News Section** | Deferred to Phase 2. If the chosen CMS makes blog functionality trivial to add (e.g., Sanity with article schema), it may be included at build time but content strategy is not part of Phase 1 scope. |
| 6 | **Online Proofing / Design Tool** | No browser-based design tool, no online artwork editor. Customers upload print-ready files. |
| 7 | **Live Order Tracking** | No real-time order tracking portal. Updates provided via WhatsApp. |
| 8 | **Customer Chat History** | No persistent chat history or CRM integration in Phase 1. Leads are tracked manually via WhatsApp/email. |
| 9 | **Mobile App** | No native mobile application. The website is fully responsive and WhatsApp-integrated. |
